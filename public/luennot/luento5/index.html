Tapaamiskerta 5: projektit Tradosissa

Teoriaa

Kahdenlaisia muisteja!

Main translation memory ("äitimuisti")

  • Oletuksena
  • Käytetään suoraan käyttäjän omistuksessa olevaan muistia, joka käännöksen jälkeen sisältää
  • Kaiken, mitä siellä oli ennen käännöstä
  • Sen, mitä sinne tallentui tätä käännöstä tehdessä

Project translation memory ("lapsimuisti")

  • Pitää erikseen valita
  • Haetaan mätsejä ja konkordansseja suoraan käyttäjän omistuksessa olevasta muistista
  • Sisältää
  • Sen, mitä sinne tallentui tätä käännöstä tehdessä
